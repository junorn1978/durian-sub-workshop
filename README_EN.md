# HamuHamu Subtitle Atelier

A real-time subtitle + translation overlay tool originally made for VTuber [Otsuka Ray](https://www.youtube.com/@OtsukaRay). All features are free for anyone to use, except the Otsuka Ray–specific filtering.

Current version: `v2.17.2`

- 🎙️ **Speech Recognition**: Browser built-in (free) / Deepgram (paid) / Soniox (high-accuracy, paid)
- 🌐 **Translation**: Google (GAS) / Custom Link / Browser built-in Translation API / Built-in AI
- 📺 **Livestream integration**: Auto-configure subtitle sources via OBS WebSocket
