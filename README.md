# ハムハム字幕アトリエ

VTuber [乙夏れい](https://www.youtube.com/@OtsukaRay) さん向けに作られた、リアルタイム字幕＋翻訳の表示ツールです。乙夏れいさん専用フィルタ機能を除き、どなたでも自由に利用できます。

現在のバージョン：`v3.1.8`

- 🎙️ **音声認識**：ブラウザ内蔵 (無料) / Soniox (高精度・有料)
- 🌐 **翻訳**：Google 翻訳 (無料) / カスタムリンク (翻訳クラウド接続)
- 📺 **配信連携**：OBS WebSocket による字幕ソース自動構築

## 🚀 すぐに使う

Web アプリを開く：**[https://junorn1978.github.io/durian-sub-workshop/](https://junorn1978.github.io/durian-sub-workshop/)**

## 📖 設定方法

各機能（翻訳クラウド接続・音声認識エンジン・OBS 連携など）の設定方法は、アプリ内の各項目にある「?」アイコンから確認できます。
