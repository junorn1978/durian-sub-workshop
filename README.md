# ハムハム字幕アトリエ

本軟體為 VTuber [乙夏れい](https://www.youtube.com/@OtsukaRay) 所製作的即時字幕＋翻譯顯示工具。除了乙夏れいさん專用過濾文字以外，其他功能皆可任意使用。

目前版本：`v2.17.2`

- 🎙️ **語音辨識**：瀏覽器內建 (免費) / Deepgram (付費) / Soniox (高精度・付費)
- 🌐 **翻譯**：Google (GAS) / 自訂連結 / 瀏覽器內建翻譯 API / 內建 AI
- 📺 **直播整合**：OBS WebSocket 自動構築字幕來源
