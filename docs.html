ハムハム字幕アトリエ

▶ ⌨

📌 概要

本ツールは、VTuber 乙夏れいさん向けに制作された、リアルタイム字幕＋翻訳の表示ツールです。乙夏れいさん専用のフィルタ機能を除き、どなたでもご自由にご利用いただけます。

音声をリアルタイムで文字起こしし、その場で翻訳して表示します。OBS で本ページのウィンドウをキャプチャすれば、配信画面にそのまま翻訳字幕を載せることができます。

🎙️ 音声認識エンジン

ブラウザ内蔵 (Web Speech API) 無料 既定

Chrome / Edge に搭載されている音声認識 API を利用します。ネットワーク経由で Google のサーバーへ音声を送って認識する方式で、追加のセットアップは不要です。手軽さ重視の方におすすめ。

Deepgram 有料

クラウド音声認識サービス。中精度。新規登録で 200 米ドル相当 のクレジット付与あり、目安は 約 1 米ドル / 時間。設定方法は Deepgram 設定ガイド を参照。

Soniox 有料

高精度の WebSocket ストリーミング音声認識サービス。日本語の認識精度に優れ、句読点や話者区切りの判定も自然です。目安は 約 1.5 米ドル / 時間。設定方法は Soniox 設定ガイド を参照。

※ 既定では「ブラウザ内蔵」が選択されています。配信品質を上げたい場合は Deepgram / Soniox の利用を検討してください。

🌐 翻訳モード

  • Google 翻訳 (内蔵) 既定：内蔵された Google 翻訳を利用（無料、設定不要）
  • 翻訳クラウド接続 (カスタムリンク)：自作のバックエンドと連携（所定フォーマット対応であれば利用可能）
  • Google Script Apps (GAS)：Google Apps Script の無料翻訳 API を利用（1 日 5,000 文まで）
  • 高速翻訳：ブラウザ内蔵のオンデバイス翻訳 API を利用（高速、精度は今後改善予定）
  • ブラウザ AI：未公開（ブラウザ内蔵 AI モデルが安定したら公開予定）

GAS の設定方法は 接続設定ガイド を参照してください。

🚀 使い方

  1. 上の「ツールを開く」から Web アプリを起動
  2. 音声認識を、配信で話す主な言語（例：日本語）に設定
  3. 翻訳先を、字幕として表示したい言語（例：英語）に設定
  4. 翻訳モードを選択（初心者は GAS 翻訳がおすすめ）
  5. 開始ボタンで録音を開始

📺 OBS との連携

OBS WebSocket 経由で字幕ソースを自動構築できます。詳しくは OBS 連携ガイド を参照してください。

🧩 対応ブラウザ

  • Chrome：v145 以上
  • Edge：v145 以上

🔧 上級者向け：自作翻訳 API との連携

自作の翻訳バックエンド（例：TranslationGemma など）と連携する場合、下記のリクエスト形式に対応していれば利用できます。

リクエスト

  • Method：POST
  • Content-Type：application/json
  • Header (Optional)：X-API-Key: <Your-API-Key>（URL に key:// プロトコルが含まれる場合は自動で付与）

Payload 例

{
  "text": "翻訳するテキスト",
  "targetLangs": ["en", "ja"],
  "sourceLang": "zh-TW",
  "sequenceId": 123,
  "previousText": "前の文脈（オプション）"
}

📌 介紹

本軟體為 VTuber 乙夏れい 所製作的即時字幕＋翻譯顯示工具。除了乙夏れいさん專用過濾文字以外，其他功能皆可任意使用。

主要功能為將語音即時轉為文字逐字顯示，並進行即時翻譯。透過 OBS 擷取本軟體視窗，即可在直播畫面上呈現翻譯字幕的效果。

🎙️ 語音辨識引擎

瀏覽器內建 (Web Speech API) 免費 預設

使用 Chrome / Edge 內建的語音辨識 API。音訊會經由網路傳送到 Google 伺服器辨識，無須額外設定，最方便上手。

Deepgram 付費

專業語音辨識服務商。中精度。新帳號註冊通常提供 $200 美金額度，預估費用 約 $1 美金 / 小時。設定方式請見 Deepgram 設定指南。

Soniox 付費

高精度的 WebSocket 串流語音辨識服務。日語辨識精度高，標點與句讀判斷自然。預估費用 約 $1.5 美金 / 小時。設定方式請見 Soniox 設定指南。

※ 預設選用「瀏覽器內建」。若需提升直播字幕品質，可考慮 Deepgram / Soniox。

🌐 翻譯模式

  • Google 翻譯 (內建) 預設：使用內建的 Google 翻譯（免費，無需任何設定）
  • 翻譯雲端連線 (自訂連結)：連接自有的翻譯後端（符合指定格式即可使用）
  • Google Script Apps (GAS)：使用 Google Apps Script 的免費翻譯 API（每日 5,000 句上限）
  • 高速翻譯：使用瀏覽器內建的裝置端翻譯 API（速度快，精度持續改善中）
  • 瀏覽器 AI：尚未開放（待瀏覽器內建 AI 模型穩定後開放使用）

GAS 的設定方式請見連線設定指南。

🚀 使用方式

  1. 從上方「開啟工具」進入網頁應用
  2. 語音辨識設為直播主要使用的語言（例：日文）
  3. 翻譯目標選擇要顯示為字幕的語言（例：英文）
  4. 翻譯模式依需求選擇（初次使用建議用 GAS 翻譯）
  5. 按下開始進行錄音

📌 Overview

This tool is a real-time subtitle + translation overlay originally made for VTuber Otsuka Ray. All features are free for anyone to use, except the Otsuka Ray–specific filtering.

It transcribes speech in real time and translates it instantly. Capture the app window in OBS to display translated subtitles on your livestream.

🎙️ Speech Recognition Engines

Browser Built-in (Web Speech API) Free Default

Uses the built-in speech recognition API in Chrome / Edge. Audio is sent over the network to Google's servers for recognition. No extra setup required — the easiest option.

Deepgram Paid

Professional cloud speech recognition. Mid-accuracy. New accounts typically receive $200 in credits. Estimated cost: ~$1 USD / hour. See Deepgram setup guide.

Soniox Paid

High-accuracy WebSocket streaming speech recognition service. Excellent at Japanese, with natural punctuation and sentence boundary detection. Estimated cost: ~$1.5 USD / hour. See Soniox setup guide.

Note: "Browser Built-in" is selected by default. For higher quality subtitles, consider Deepgram or Soniox.

🌐 Translation Modes

  • Google Translation (Built-in) Default: Uses the built-in Google translation (free, no setup required)
  • Translation Cloud (Custom Link): Connect your own backend (if it matches the required request format)
  • Google Apps Script (GAS): Uses Google Apps Script's free translation API (limit: 5,000 sentences/day)
  • High-speed Translation: Uses the browser's built-in on-device translation API (fast, accuracy still improving)
  • Browser AI: Not yet available (will be enabled once the browser built-in AI model stabilizes)

For GAS setup details, see the connection setup guide.

🚀 How to use

  1. Click "Open the app" above to launch
  2. Set Speech Recognition to your main streaming language (e.g., Japanese)
  3. Set Target language to the language you want shown as subtitles (e.g., English)
  4. Pick a Translation Mode (GAS Translation is recommended for first-time users)
  5. Press Start to begin recording

💡 Recommended: open in a standalone window

🔧 Advanced: Custom translation backend

If you want to use a self-hosted translation API (e.g., TranslationGemma), you can connect it as long as the backend supports the request format below.

Request

  • Method: POST
  • Content-Type: application/json
  • Header (Optional): X-API-Key: <Your-API-Key> (automatically included if the URL uses the key:// protocol)

Payload example

{
  "text": "Text to translate",
  "targetLangs": ["en", "ja"],
  "sourceLang": "zh-TW",
  "sequenceId": 123,
  "previousText": "Previous context text (optional)"
}
